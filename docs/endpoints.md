# API контракт

Базовый URL: `http://localhost:8000`

Документация (Swagger UI): `http://localhost:8000/docs`

🔒 — эндпоинт требует валидный JWT-токен в заголовке `Authorization: Bearer <token>`

## Аутентификация

| # | Метод | URL | Описание |
|---|-------|-----|----------|
| 1 | POST | `/auth/register` | Регистрация нового пользователя |
| 2 | POST | `/auth/login` | Логин, возвращает JWT-токен |
| 3 | GET | `/auth/me` 🔒 | Данные текущего пользователя |

## Инициативы

| # | Метод | URL | Описание |
|---|-------|-----|----------|
| 4 | GET | `/initiatives/` | Список инициатив с фильтрами по статусу и категории |
| 5 | GET | `/initiatives/{id}` | Детали инициативы + список оценок |
| 6 | POST | `/initiatives/` 🔒 | Создание новой инициативы |
| 7 | PUT | `/initiatives/{id}` 🔒 | Обновление (только автор) |
| 8 | DELETE | `/initiatives/{id}` 🔒 | Удаление (только автор) |

## Оценки экспертов

| # | Метод | URL | Описание |
|---|-------|-----|----------|
| 9 | GET | `/initiatives/{id}/estimates/` | Все оценки инициативы |
| 10 | POST | `/initiatives/{id}/estimates/` 🔒 | Добавить свою оценку по RICE |
| 11 | PUT | `/estimates/{id}` 🔒 | Изменить свою оценку |
| 12 | DELETE | `/estimates/{id}` 🔒 | Удалить свою оценку |

## Категории

| # | Метод | URL | Описание |
|---|-------|-----|----------|
| 13 | GET | `/categories/` | Список всех категорий |
| 14 | POST | `/categories/` 🔒 | Создать категорию (только admin) |
| 15 | POST | `/initiatives/{id}/categories/{cat_id}` 🔒 | Привязать категорию к инициативе |

## Системные эндпоинты

| Метод | URL | Описание |
|-------|-----|----------|
| GET | `/health` | Проверка работоспособности сервиса |
| GET | `/docs` | Swagger UI (автогенерация FastAPI) |
| GET | `/openapi.json` | OpenAPI-спецификация |

## Коды ответов

| Код | Описание |
|-----|----------|
| 200 | Успешный запрос |
| 201 | Ресурс создан |
| 204 | Успешное удаление (без тела ответа) |
| 400 | Некорректный запрос |
| 401 | Не авторизован (отсутствует или невалидный токен) |
| 403 | Нет прав (например, не автор ресурса) |
| 404 | Ресурс не найден |
| 409 | Конфликт (например, email уже зарегистрирован) |
| 422 | Ошибка валидации данных |
